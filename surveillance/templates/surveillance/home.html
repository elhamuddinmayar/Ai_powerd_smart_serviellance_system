System Overview
About Butterfly
Butterfly is a closed-circuit AI surveillance platform built for security operations centres. It integrates real-time computer vision, facial recognition, gesture detection and a structured incident-management workflow into a single unified interface.

The system is designed for multi-role teams: admins configure and control, supervisors upload targets and verify detections, operators monitor assigned targets and submit field reports.
Django 5 Daphne ASGI Channels WS Redis OpenCV YOLO11 DeepFace Facenet512 ReportLab MySQL Leaflet.js
Core Capabilities
  • Live MJPEG camera streams with per-frame JPEG polling — compatible with any ASGI server
  • YOLO11-Pose person counting and gesture classification (fall detection, hand waving)
  • DeepFace facial recognition with configurable cosine-distance threshold and match cooldown
  • Real-time WebSocket broadcast of alarms, matches and camera status changes
  • Interactive Leaflet map with camera pin placement and live status markers
  • Full assignment workflow: enroll → assign → acknowledge → detect → verify → export
  • Role-scoped data access: admin / supervisor / operator with per-view guards
  • PDF dossier and detection report generation with reference photos and evidence snapshots
  • Multi-language support (English, Persian, Pashto) with RTL layout switching
  • Auto-discovery of local camera indices with accessibility probing
